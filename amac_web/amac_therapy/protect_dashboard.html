How to Protect Your AMAC Therapy Dashboard

Option 1: Add this to your dashboard HTML (before </body>)

<script> // Login check for AMAC Therapy (function() { const isLoggedIn = localStorage.getItem('amacLoggedIn') || sessionStorage.getItem('amacLoggedIn'); if (!isLoggedIn) { window.location.href = 'login.html'; } // Add logout button const logoutBtn = document.createElement('button'); logoutBtn.textContent = 'Logout'; logoutBtn.style.cssText = \ position: fixed; top: 20px; right: 20px; background: #2c5aa0; color: white; border: none; padding: 10px 20px; border-radius: 6px; cursor: pointer; z-index: 1000; font-weight: bold; \; logoutBtn.onclick = function() { localStorage.clear(); sessionStorage.clear(); window.location.href = 'login.html'; }; document.body.appendChild(logoutBtn); })(); </script>

Option 2: Use a meta redirect in your dashboard

<!-- Add this at the very top of your dashboard HTML --> <script> if (!localStorage.getItem('amacLoggedIn') && !sessionStorage.getItem('amacLoggedIn')) { window.location.href = 'login.html'; } </script>

Steps:

  1. Open your dashboard HTML file (probably index.html or similar)
  2. Add Option 1 code just before </body> tag
  3. Save and refresh
  4. Now users must login first!
