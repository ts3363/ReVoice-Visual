# AMAC Therapy System

Adaptive Multimodal Assistive Communication and Speech Training System for dysarthric speech.

## Features
- Real-time speech therapy with audio/video analysis
- Adaptive exercises with progressive difficulty
- Progress tracking and visualization
- Multimodal feedback system

## Quick Start
1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`
4. Open: http://localhost:8000/static/therapy.html
